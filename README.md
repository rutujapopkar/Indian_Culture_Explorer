# 🗺️ Indian Culture Explorer

An interactive, gamified React web application designed to explore, map, and celebrate India's protected **Geographical Indication (GI)** treasures. Users can engage in an educational puzzle game, dragging unique regional specialties onto their native states to unlock rich cultural histories, official registration details, and authentic imagery.

---

## 🚀 Key Features

* **Cinematic Introduction:** An immersive landing page featuring a full-screen video backdrop to set an engaging cultural tone before the experience begins.
* **Gamified Drag-and-Drop Mechanics:** An intuitive, interactive sidebar grid where users select regional GI icons (such as *Paithani Silk*, *Bebinca*, and *Panchmura Terracotta*) and match them with their respective states.
* **Real-Time Map Feedback:** Instant boundary success tracking using interactive HTML layout coordinates linked directly to a map sourced from the *Survey of India*.
* **Deep-Dive Knowledge Overlays:** Correctly mapping an item reveals a sleek informational modal highlighting the product's official category, GI registration number, and regional historical context accompanied by crisp, high-resolution imagery.

---

## 🛠️ Tech Stack & Architecture

* **Frontend Library:** React.js (Functional components, custom Hooks for state and drag-drop event routing)
* **Styling & Layout:** Custom CSS3 (Flexbox/Grid architecture with relative percentage-based positioning for responsive map tracking)
* **Assets:** Custom vector boundaries (SVG) and dynamic MP4 video elements.

## 📦 Getting Started

To run this project locally, follow these quick setup steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Installation
Clone the repository and install the dependencies:
```bash
git clone [https://github.com/YOUR_USERNAME/indian-culture-explorer.git](https://github.com/YOUR_USERNAME/indian-culture-explorer.git)
cd indian-culture-explorer
npm install

### 3. Available Scripts

In the project directory, you can run:

    npm start Runs the app in development mode. Open http://localhost:3000 to view it in your browser. The page will auto-reload when you make code modifications.

    npm run build Builds the application for production to the build folder, correctly bundling and optimizing React for peak performance.
